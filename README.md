# front

## Deploy
http://3.34.219.42:3000

## Pictogram
Picktogram은 디자이너들이 자유롭게 소통할 수 있도록 설계된 SNS 서비스입니다. (개발중)

## Skill
* ReactJS
* NextJS
* Typescript
* Emotion
* Nestia SDK를 이용한 통신 (백엔드와의 연동 방식)

## 기능
* 간편 회원가입
* 로그인
* 게시글 작성 (게시글에 타입을 설정하여 목적에 따른 게시글 형태를 제공할 예정입니다.)
* 댓글 작성
